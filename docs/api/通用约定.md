# API 通用约定

> 所有接口统一遵守以下约定，详细接口文档参见 [docs/api/](../api/) 目录。

---

## 1. 统一前缀

```
/api/v1
```

## 2. 统一返回结构

所有接口统一返回：

```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

| 字段 | 说明 |
|------|------|
| `code = 0` | 成功 |
| `code != 0` | 失败 |
| `message` | 提示信息 |
| `data` | 业务数据 |

## 3. 分页结构

列表接口返回分页结构：

```json
{
  "items": [],
  "total": 100,
  "page": 1,
  "page_size": 20
}
```

## 4. 认证方式

- `Authorization: Bearer <access_token>`
- `tenant_id` 不由前端直接传，默认从登录态 JWT 中解析
